# Erlang-Concurrent-TCP-Server - Example 1

## Compilation

```shell
erl -make
```

## Running server

### Windows

```shell
_server_run.bat
```

### Linux
```shell
./_server_run.sh
```
## Running terminal (worker)

### Windows

```shell
_terminal_run.bat
```

### Linux
```shell
./_terminal_run.sh
```
